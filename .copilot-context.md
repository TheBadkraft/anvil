# Document Standards — anvil
- Standards: [q-or/DOC_STANDARDS.md](../q-or/DOC_STANDARDS.md)
- Registry:  [q-or/registry.anvl](../q-or/registry.anvl)
- This project's doc root: `anvil/bug-reports/`, `anvil/feature-reqs/`, `anvil/tickets/`
- Cross-project link syntax: `../other-project/subdir/filename.md`
- To publish a doc for cross-project use: add entry to registry.anvl first
- ID prefix for new bugs: `BR-{YYMM}-anvil-{seq:03d}`
- ID prefix for new features: `FR-{YYMM}-anvil-{seq:03d}`
## Docs
- Register a new document: `python3 ../tools/doc-register.py --id <ID> --type <type> --owner anvil --path "anvil/subdir/file.md" --title "..." --status open --tags "tag1,tag2"`
- `--type` values: `bug-report`, `feature-req`, `ticket`, `use-case`, `backlog`
- Or edit [q-or/registry.anvl](../q-or/registry.anvl) directly; full conventions: [q-or/DOC_STANDARDS.md](../q-or/DOC_STANDARDS.md)
- Dependency state: check [q-or/repo.anvl](../q-or/repo.anvl) before starting feature work
## Ops
### Build
```
cbuild           # compile sources (default)
cbuild lib       # compile + link shared lib
cbuild install   # compile + link + install to /usr/local/packages/
cbuild clean     # remove build/ artifacts
cbuild clean_all # remove build/ + bin/ artifacts
```

### Test
```
./rtest                    # run all tests
./rtest unit               # all tests in test/unit/
./rtest unit/<name>        # run test_<name>.c only (e.g. unit/parser)
./rtest --config <key>     # keys: resolver, serializer, vars, import, asl, schema, prototype
```

### Packages
```
cpkg anvil             # core parser (anvil.o)
cpkg resolver          # anvil.resolver.o
cpkg anvil.serializer  # anvil.serializer.o
cpkg anvil.vars        # anvil.vars.o
cpkg anvil.import      # anvil.import.o
cpkg anvil.asl         # anvil.asl.o
cpkg anvil.schema      # anvil.schema.o
```

### Dependencies
- Runtime: `sigma.memory`, `sigma.text`
- Test:    `sigma.memory`, `sigma.text`, `sigma.test`
- cbuild/rtest/cpkg are at `/usr/local/scripts/`; `./rtest` is the local per-project wrapper
